# 建模假设、稳健性与局限说明

## 1. 基本假设

本项目基于原报告的计量经济学框架，默认以下假设成立：

1. 年度数据口径在样本期内具有可比性。
2. 对数变换后的变量能较好刻画比例变化关系。
3. 净利润与营业收入、营业成本、GDP 增长率和 CPI 之间可能存在长期均衡关系。
4. 若长期残差平稳，则可使用 ECM 分析短期调整机制。
5. 回归残差在最终模型中不存在严重异方差和自相关问题。

## 2. 对数变换假设

对数模型隐含变量之间的弹性关系。例如 `DLNX2` 的系数可近似解释为营业收入增长 1% 时，净利润增长率变化多少个百分点。

由于所有原始变量均为正数，因此可以直接取自然对数。

## 3. 协整假设

ECM 的前提是变量之间存在长期均衡关系。原报告通过 Engle-Granger 两步法进行检验：

1. 用 `LNY` 对 `LNX2`、`LNX3`、`LNX4`、`LNX5` 做长期 OLS 回归。
2. 对长期回归残差做 ADF 检验。
3. 若残差平稳，则认为存在协整关系。

## 4. 诊断检验解释

| 检验 | 目的 | 原报告结论 |
|---|---|---|
| VIF | 判断多重共线性 | 未发现严重多重共线性 |
| White | 判断异方差 | 不拒绝同方差原假设 |
| Breusch-Godfrey LM | 判断残差自相关 | 不拒绝无自相关原假设 |

## 5. 数据与模型局限

### 样本量较小

样本区间为 2000-2023 年，共 24 个年度观测值。对于时间序列模型而言，样本量偏小，统计检验结果可能对滞后阶数和趋势项设定敏感。

### 宏观变量频率较低

年度 GDP 增长率和 CPI 可能无法充分反映消费升级、行业景气度、白酒价格政策、渠道库存和公司治理等更直接影响贵州茅台利润的因素。

### 变量可能存在强趋势

净利润、营业收入、营业成本和 CPI 都具有明显趋势。虽然协整和 ECM 可以处理同阶单整变量，但仍需要谨慎解释长期系数。

### 因果解释需要谨慎

本项目主要是时间序列相关性和协整关系分析，并不能单独证明严格因果关系。若要进行因果识别，需要更丰富的研究设计，例如工具变量、事件研究或面板数据方法。

## 6. 可扩展方向

1. 将年度数据扩展为季度数据，提高样本量。
2. 加入白酒行业景气指数、贵州茅台产品出厂价、销量、预收款/合同负债等公司经营变量。
3. 使用 ARDL、VAR/VECM 与滚动窗口检验进行稳健性比较。
4. 将模型扩展为同业公司面板数据，比较贵州茅台与其他白酒企业的利润驱动因素差异。
